# Go Event Management API

A simple REST API built with Go and Gin for user authentication, event management, and event registration.

## Features

- User sign up and login
- JWT-based authentication
- Public event listing and event detail endpoints
- Authenticated event CRUD (create/update/delete)
- Register and unregister for events
- SQLite database with auto-created tables

## Testing Requests

You can run ready-made HTTP request files from the `api-test/` folder (for example in VS Code REST Client):

- `register.http`
- `login.http`
- `crease-uuer.http`
- `create-event.http`
- `get-events.http`
- `get-event.http`
- `update-event.http`
- `delete-event.http`
- `unregister.http`
